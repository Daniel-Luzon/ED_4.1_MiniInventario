# MiniInventario
Aplicación de consola en Java para practicar Git y GitHub.
## Requisitos
Java
IntelliJ IDEA
Git
## Ejecución
Abrir el proyecto en IntelliJ IDEA y ejecutar la clase Main.

## Nota de verificación
Este cambio se ha realizado desde una copia clonada del repositorio...

## Uso del programa
Ejecuta MiniInventario.
Elige entre añadir, listar o buscar productos.
La nueva opción de búsqueda permite localizar un producto por nombre.
Si el producto existe, el sistema lo indica por pantalla.
